# Contributing a paper to PaperStack

Thanks for helping grow this. Here's how to add a question paper:

1. **Fork** this repository (button top-right on GitHub).
2. Clone your fork locally, or just edit directly on GitHub.com.
3. Go to `public/papers/<COURSE>/` — pick the matching course folder (`Btech`, `BCA`, `BBA`, `MCA`).
4. Add your PDF, named like: `subjectname-sem3-2023-endsem.pdf`
   (lowercase, no spaces, use hyphens)
5. Commit your change with a message like: `Add Data Structures Sem3 2023 paper`
6. Open a **Pull Request** back to this repo.
7. I'll review and merge it. That's it.

If you're not comfortable with Git/GitHub, use the no-login submission form linked on the site instead.
